# AI Document Q&A (RAG Pipeline)

A simple RAG pipeline using Streamlit and Gemini API.  
Upload documents (PDF, DOCX, TXT) and ask questions about their content.

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add `.env` file with:
GEMINI_API_KEY=your_api_key_here
4. Run the app:
streamlit run app.py
